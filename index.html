HTML (Hypertext Markup Language)

  • Think of HTML as the bricks and structure of a house.
  • The <head> tag contains information about the webpage.
  • The <body> tag contains the content and the structure of the webpage.
  • HTML uses angel brackets (<>) for the tags.
  • When using HTML, try to use semantic coding. This will help who require assistance (such as screenreaders) to navigate your website.
