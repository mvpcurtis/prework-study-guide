HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an elemant.
  • A padding indicates how much space we want around the content inside of an element.
  • A border sits between the padding and the margin elemants.
  • All of these together is called the "box model".

GIT

  • git status: checks what branch we are currently on.
  • git checkout -b *branch-name*: create a new branch and switches to it.
  • git checkout *branch-name*: switches you to that branch.
  • git add -A: adds the changes in our files/repos to the staging area of Git.
  • git commit -m "add note to project here": commits our changes.
  • git pull origin (branch): pulls the latest updates to the branch down to your computer.
  • git push origin (branch): pushes all changes to GitHub.

JavaScript

  • Control flow: the order in whihch a computer executes code in the script. However, it can be interrupted by conditional statements and loops.
  • Conditional statements: a way of introducing decision-making into our code.
  • Arrays: we use these to store a group of related data.
  • Varibles: a named container that allows us to store data in our code.
  • For loops: used to perform a task multiple times.
  • Functions: they do not execute on their own. They have to be called!
  • Math.floor() & Math.random() are both properties of JavaScripts built-in Math object that can be used together to generate a random number.
